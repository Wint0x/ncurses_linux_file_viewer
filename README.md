# Requirements  
C++ 20  
ncurses  

# Compile  
```sh  
make  
```  
Or  
```  
g++ file_viewer.cpp -o file_viewer.out -std=c++20 -lncurses
```  
# Run  
```sh  
./file_viewer.out {specify_path}  
```
